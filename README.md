## quick start

- Install [docker](https://docs.docker.com/engine/install/)
- Run in terminal `$ docker-compose up -d `
- description of REST API [http://localhost:3000/api/](http://localhost:3000/api/)
- defolt user
``` 
username : super.admin
password : super.admin
```



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
